
# Maniac Mansion-inspired text adventure for Godot 4



## What's Included:

1.  **ManiacMansionGame.gd** - The core game logic with:
    -   Multiple rooms to explore (driveway, mansion, basement, laboratory, etc.)
    -   Inventory system
    -   Puzzle mechanics (finding keys, feeding tentacles, rescuing Sandy)
    -   Game state tracking
2.  **main.gd** - The test runner UI controller
3.  **main.tscn** - Complete scene file with a retro-styled interface

## How to Set Up in Godot 4:

1.  Create a new Godot 4 project
2.  Create three files:
    -   `ManiacMansionGame.gd` (first artifact)
    -   `main.gd` (second artifact)
    -   `main.tscn` (third artifact)
3.  Run the scene!

## How to Play:

**Commands:**

-   `look` - Examine your surroundings
-   `go north/south/east/west/up/down` - Move between rooms
-   `take [item]` - Pick up items
-   `use [item]` - Use items from inventory
-   `inventory` - Check what you're carrying
-   `examine [item]` - Look at items closely

**Quick Walkthrough:**

1.  Take the flashlight from the driveway
2.  Go north to the porch, take the doormat
3.  Use the doormat to find the rusty key
4.  Use the rusty key to unlock the front door
5.  Explore the mansion, collect the pepperoni and wrench
6.  Use the wrench in the basement to open the lab door
7.  Take the lab key in the laboratory
8.  Use the lab key to rescue Sandy and win!

The game captures the spirit of the original with mansion exploration, item puzzles, and the goal of rescuing your friend from Dr. Fred's laboratory!
