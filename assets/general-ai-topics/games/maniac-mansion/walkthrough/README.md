
# MANIAC MANSION - Complete Walkthrough

## Objective

Rescue your friend Sandy from Dr. Fred's mansion laboratory!

----------

## Complete Solution (Fastest Route)

### Step 1: Starting Area - Driveway

```
take flashlight
go north

```

### Step 2: Front Porch - Get Inside

```
take doormat
use doormat
(You'll find a rusty key)
use rusty_key
(This unlocks the front door)
go north

```

### Step 3: Explore the Mansion - Kitchen

```
go west
take pepperoni
go east

```

### Step 4: Living Room - Find First Key

```
go east
take old_key

```

### Step 5: Basement - Get the Wrench

```
go down
take wrench
use wrench
(This forces open the laboratory door)

```

### Step 6: Enter the Laboratory

```
go north
take lab_key
use lab_key

```

**GAME WON!** You've rescued Sandy!

## Optional Content

### Feed the Purple Tentacle

If you want to interact with the Purple Tentacle in the backyard:

```
(After getting pepperoni from kitchen)
go north
use pepperoni

```

The tentacle will be happy, but this isn't required to win.

### Explore Upper Floor

From the foyer:

```
go up
go east

```

You'll find Green Tentacle's room with a record player.

## Tips & Tricks

-   **You don't need every item to win** - Only the doormat, wrench, and lab key are essential
-   **The fastest route** takes about 12 commands total
-   **No death scenarios** - You can't lose or die in this version
-   **Commands are flexible** - "go north" = "go n" = "move north"
-   **Use Tab/Arrow keys** - Makes typing commands faster

----------

## Command Reference

### Movement

-   `go north` (or n, south, east, west, up, down)
-   `look` - Examine current room

### Items

-   `take [item]` - Pick up item
-   `use [item]` - Use item from inventory
-   `examine [item]` - Get item description
-   `inventory` (or `i`) - List what you're carrying

### Other

-   `help` - Show command list
